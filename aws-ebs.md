## Elastic Block Store (EBS)
Complete a Snapshot
```shell
aws ebs complete-snapshot (snapshot-id)
```

Start a Snapshot
```shell
aws ebs start-snapshot --volume-size (value)
```

Get a Snapshot block
```shell
aws ebs get-snapshot-block
```

```shell
--snapshot-id (value)
```

```shell
--block-token (value)
```
